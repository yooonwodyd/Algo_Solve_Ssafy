# 거울에 레이저 쏘기 2

- **XP**: 70XP  
- **평균 소요 시간**: 180분  
- **정답률**: 43%  
- **총 제출 수**: 3,662회  

---

## 🧩 문제 설명

N × N 크기의 격자 안에 각 칸마다 거울이 하나씩 들어 있습니다.  
각 거울은 `\` 또는 `/` 형태입니다.

격자 밖에는 레이저를 쏠 수 있는 위치가 총 **4N개** 있으며,  
그 중 특정 위치에서 레이저를 쐈을 때 **거울에 튕기는 횟수(반사 횟수)** 를 구하는 프로그램을 작성하세요.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N**이 주어집니다.
2. 두 번째 줄부터 **N개의 줄**에 걸쳐 격자의 거울 정보가 주어집니다.  
   - 각 줄은 공백 없이 **N개의 문자**로 이루어져 있으며, 문자는 `\` 또는 `/` 입니다.
3. 다음 줄에 레이저를 쏘는 위치를 나타내는 정수 **K**가 주어집니다.

### 🔢 레이저 시작 위치 K의 의미

레이저는 격자 바깥의 **4N개 위치** 중 하나에서 시작합니다.  
가능한 시작 위치를 **시계방향 순서대로** 1부터 4N까지 번호를 붙였을 때,  
입력으로 주어지는 **K**는 그 번호입니다.

(그림 예시처럼, 위쪽 가장자리(1~N) → 오른쪽(N+1~2N) → 아래쪽(2N+1~3N) → 왼쪽(3N+1~4N) 순서)

---

## 📤 출력

- 주어진 위치에서 레이저를 쐈을 때, **거울에 튕기는 횟수**를 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 1000\)
- \(1 \le K \le 4N\)

---
