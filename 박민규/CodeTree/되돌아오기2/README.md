# 되돌아오기 2

- **XP**: 20XP  
- **평균 소요 시간**: 18분  
- **정답률**: 69%  
- **총 제출 수**: 3,481회  

---

## 🧩 문제 설명

좌표평면 위에서 **(0, 0)** 에서 시작하며, 처음에는 **북쪽**을 바라보고 있습니다.  
문자열로 주어지는 **N개의 명령**을 순서대로 수행하며 총 **N번** 움직입니다.

명령은 다음 3가지로 이루어집니다.

- **L**: 왼쪽으로 90도 회전
- **R**: 오른쪽으로 90도 회전
- **F**: 바라보고 있는 방향으로 **1칸 전진**

시간 규칙은 다음과 같습니다.

- **전진(F)**: 1초에 1칸 이동 → **1번 명령 수행 = 1초**
- **회전(L/R)**: 회전에도 **1초 소요** → **1번 명령 수행 = 1초**

이때, 명령을 수행하는 도중 **처음으로 다시 (0, 0)** 에 도착하는 시점이 몇 초 뒤인지 출력하세요.  
만약 **N번 명령을 모두 수행할 때까지** (0, 0)으로 돌아오지 못하면 **-1**을 출력합니다.

---

## 📥 입력

- 첫 번째 줄에 문자 **L, R, F**로만 이루어진 문자열 형태로 명령이 주어집니다.

---

## 📤 출력

- 처음으로 다시 **(0, 0)** 으로 되돌아오는 데 걸리는 시간을 출력합니다.
- 끝까지 돌아오지 못하면 **-1**을 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le \text{명령의 길이} \le 100{,}000\)
- 명령은 문자 **L, R, F**로만 이루어진 문자열입니다.
- **첫 명령은 항상 F**로 주어집니다.

---

## ✅ 입력 예제 1
```
FFFRRFFRFFFRFFFFF
```

## ✅ 출력 예제 1
```
13
```
