# 폐허 조사

- **정답률**: 28%  
- **총 제출 수**: 7회  

---

## 🧩 문제 설명

탐험가 코디는 폐허가 된 도시를 조사합니다. 폐허는 **N × N 격자 지도**로 표현되며,
각 칸에는 **오염 수치**가 적혀 있습니다.

코디는 **정신력(M)** 을 가지고 있으며, 어떤 칸으로 **이동할 때마다**
이동한 칸의 오염 수치만큼 정신력이 감소합니다.

- 정신력이 **0 이하**가 되면 코디는 **더 이상 움직일 수 없습니다.**
- 단, **마지막 이동으로 정신력이 0 이하가 되는 경우는 허용**합니다.  
  (즉, 이동 계획을 전부 수행한 후 0 이하가 되어도 “최종 위치 가능”으로 봄)

코디는 (1,1)에서 시작합니다.

---

## 🚶 이동 규칙

현재 위치가 (x, y)일 때 다음 이동이 가능합니다.

- `U` : (x-1, y)
- `L` : (x, y-1)
- `R` : (x, y+1)
- `D` : (x+1, y)

단, 다음과 같은 경우에는 **지도 밖 이동**이므로 실제로 이동하지 않습니다.

- x 또는 y가 **0 이하**
- x 또는 y가 **N 초과**

이때도 해당 이동 명령은 **수행한 것으로 처리하고 건너뜁니다.**

---

## 🎯 목표

폐허 지도 정보와 코디의 이동 계획 문자열이 주어질 때,

- 코디가 **이동 계획을 전부 수행한 시점에 실제로 어떤 칸에 위치할 수 있는지**를 판단하고,
- 가능하다면 그때의 최종 위치를 출력합니다.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M**이 공백으로 주어집니다.
   - N: 폐허 격자 크기
   - M: 코디의 정신력 수치
2. 두 번째 줄에 정수 **L**이 주어집니다. (이동 계획 문자열 길이)
3. 세 번째 줄에 문자열 **S**가 주어집니다. (길이 L)
   - S는 `U, L, R, D`로만 이루어짐
4. 네 번째 줄부터 N개의 줄에 걸쳐 격자의 오염 수치 **A_i,j**가 주어집니다.
   - i번째 줄에 N개의 정수가 공백으로 주어짐

---

## 📤 출력

1. 첫 번째 줄에, 이동 계획을 전부 수행한 칸에 **최종적으로 위치할 수 있으면** `Y`,
   **불가능하면** `N`을 출력합니다.
2. 두 번째 줄에, 최종 위치의 좌표 `x y`를 공백으로 출력합니다.

---

## ⚙️ 제한 조건

- \(4 \le N \le 1000\)
- \(1 \le M \le 10^9\)
- \(1 \le L \le 100{,}000\)
- \(0 \le A_{i,j} \le 10{,}000\)
- 문자열 S는 `U, L, R, D`로만 이루어짐
- \((1,1)\)의 오염 수치는 항상 **0**

---
