## 🧩 문제 설명

A와 B는 **같은 시작점**에서 **같은 방향**으로 출발합니다.  
이동 도중 **방향을 바꾸는 일은 없고**, 각자 주어진 구간별 속도로 일정 시간 동안 이동합니다.

- A는 N개의 구간
- B는 M개의 구간

각 구간은 `(v, t)`로 주어지며,
- `v`: 속도
- `t`: 해당 속도로 이동하는 시간

목표는 이동 중 **선두(앞서 있는 사람)가 몇 번 바뀌는지** 출력하는 것입니다.

### ✅ 선두 판정 규칙
- 어떤 시점에 A가 더 앞서면 A가 선두
- B가 더 앞서면 B가 선두
- **두 사람이 같은 위치(공동 선두)** 인 경우에는  
  그 상태에서 누군가 다시 앞서기 전까지는 **선두가 바뀌지 않은 것으로** 판단합니다.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M**이 주어집니다.
2. 다음 **N줄**에는 A의 이동 정보 `(v, t)`가 주어집니다.
3. 다음 **M줄**에는 B의 이동 정보 `(v, t)`가 주어집니다.

추가로 다음이 보장됩니다.
- A가 이동한 총 시간과 B가 이동한 총 시간은 **항상 동일**합니다.

---

## 📤 출력

- 선두가 **몇 번 바뀌는지** 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N, M \le 1000\)
- \(0 \le v, t \le 1000\)

---
