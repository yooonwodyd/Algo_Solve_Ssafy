# 작은 구슬의 이동

- **XP**: 30XP  
- **평균 소요 시간**: 36분  
- **정답률**: 53%  
- **총 제출 수**: 6,677회  

---

## 🧩 문제 설명

벽으로 둘러싸인 **N × N 격자** 안에 구슬 1개가 있습니다.  
구슬은 **상/하/좌/우 중 한 방향**으로 **1초에 1칸씩** 이동합니다.

격자는 가장 왼쪽 위 칸이 **(1행, 1열)**, 가장 오른쪽 아래 칸이 **(N행, N열)** 입니다.

구슬이 **벽에 부딪히면** 다음 규칙을 따릅니다.

- 이동 방향이 **반대로 뒤집힙니다**.
- 방향을 바꾸는 데 **시간 1초가 추가로 소요**됩니다.
- 이후에도 동일한 방식으로 이동/반전을 반복합니다.

구슬의 초기 위치와 초기 방향이 주어질 때, **T초 후 구슬의 위치**를 출력하세요.

---

## 📥 입력

- 첫 줄에 두 정수 **N, T**가 공백으로 구분되어 주어집니다.
- 둘째 줄에 구슬의 초기 정보 **R, C, D**가 공백으로 구분되어 주어집니다.
  - **R, C**: 구슬의 초기 위치 (R행 C열)
  - **D**: 초기 이동 방향이며 **U, D, R, L** 중 하나
    - U: 위, D: 아래, R: 오른쪽, L: 왼쪽

---

## 📤 출력

- **T초 후** 구슬이 위치한 격자 칸의 **행 번호와 열 번호**를 공백으로 구분해 출력합니다.

---

## ⚙️ 제한 조건

- \(2 \le N \le 50\)
- \(1 \le T \le 100\)
- \(1 \le R, C \le N\)
- \(D \in \{U, D, R, L\}\)

---

## ✅ 입력 예제 1
```
4 4
1 2 L
```

## ✅ 출력 예제 1
```
1 3
```
