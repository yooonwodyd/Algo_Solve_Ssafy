# 순찰 명령

- **정답률**: 100%  
- **총 제출 수**: 2회  

---

## 🧩 문제 설명

마을의 경비병 코디는 매일 밤 주어진 명령에 따라 마을을 순찰합니다.  
명령은 `W`, `A`, `S`, `D`로만 이루어진 길이 **N**의 문자열로 주어집니다.

코디는 (0, 0)에서 시작하며, 현재 위치를 (x, y)라고 할 때 각 명령은 다음과 같이 이동합니다.

- `W` : (x-1, y) 로 이동
- `A` : (x, y-1) 로 이동
- `S` : (x+1, y) 로 이동
- `D` : (x, y+1) 로 이동

---

## 🚫 이동 불가 조건

마을에는 코디가 진입할 수 없는 **접근 금지 구역**이 K개 존재합니다.  
또한 좌표가 아래 범위를 벗어나는 곳으로는 이동할 수 없습니다.

- x 또는 y가 **음수**
- x 또는 y가 **30000 초과**

만약 순찰 명령에 따라 이동하려는 위치가 위의 이동 불가 조건에 해당하면,
해당 명령은 **건너뜁니다(무시)**.

---

## 🎯 목표

코디가 (0, 0)에서 시작하여 순찰 명령을 모두 처리한 뒤,
최종 위치 (x, y)를 출력하세요.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, K**가 공백으로 주어집니다.
   - N: 순찰 명령 문자열 길이
   - K: 접근 금지 구역 개수
2. 두 번째 줄에 길이 **N**의 순찰 명령 문자열이 주어집니다.
3. 세 번째 줄부터 **K개의 줄**에 걸쳐 접근 금지 구역 좌표 `x_i y_i`가 공백으로 주어집니다.
   - 이는 코디가 해당 위치로 **진입할 수 없음**을 의미합니다.

---

## 📤 출력

- 순찰을 마친 코디의 최종 위치 `(x, y)`를 공백으로 구분하여 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 50{,}000\)
- \(1 \le K \le 100\)
- \(0 \le x_i, y_i < 30{,}000\)
- 순찰 명령 문자열은 `W`, `A`, `S`, `D`로만 이루어집니다.

---

## ✅ 입력 예제 1
```
6 2
WADDDW
1 0
0 2
```

## ✅ 출력 예제 1
```
0 1
```
