# 격자 위의 편안한 상태

- **XP**: 40XP  
- **평균 소요 시간**: 28분  
- **정답률**: 68%  
- **총 제출 수**: 3,692회  

---

## 🧩 문제 설명

N × N 크기의 격자 위에 총 M번에 걸쳐 색칠을 진행합니다.  
한 번에 **한 칸만** 색칠하며, 색칠을 한 직후 해당 칸이 **'편안한 상태'** 인지 확인하려 합니다.

### ✅ 편안한 상태의 정의
방금 막 칠해진 칸을 기준으로 **상/하/좌/우 인접한 4칸** 중에서  
**격자 밖이 아닌 칸들** 중 **색칠되어 있는 칸의 개수가 정확히 3개**인 경우를 말합니다.

> 격자 밖으로 나가는 방향은 인접 칸으로 세지 않습니다.

색칠할 칸이 주어질 때마다,  
해당 칸을 칠한 직후 **막 칠한 칸이 편안한 상태인지(1/0)** 를 출력하세요.

---

## 📥 입력

- 첫 번째 줄에 정수 **N, M**이 공백으로 구분되어 주어집니다.
- 두 번째 줄부터 **M개의 줄**에 걸쳐, 각 줄마다 색칠할 칸의 위치 **(r, c)** 가 공백으로 구분되어 주어집니다.
  - (r, c)는 r행 c열을 의미합니다.
  - **같은 칸이 2번 이상 주어지지 않습니다.**

---

## 📤 출력

- 총 **M줄**을 출력합니다.
- i번째 색칠 직후, 방금 칠한 칸이 편안한 상태이면 **1**, 아니면 **0**을 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 100\)
- \(1 \le M \le N \times N\)
- \(1 \le r, c \le N\)
