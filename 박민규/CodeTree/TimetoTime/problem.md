# Time to Time

2011년 11월 11일 A시 B분에서 시작하여  
2011년 11월 11일 C시 D분까지 몇 분이 걸리는지를 계산하는 프로그램을 작성하세요.

---

## 입력

첫 번째 줄에 A, B, C, D가 공백을 사이에 두고 주어집니다.

---

## 제한 조건

- 0 ≤ A, C ≤ 23
- 0 ≤ B, D ≤ 59
- 모순된 경우는 주어지지 않는다고 가정해도 좋습니다.

---

## 출력

첫 번째 줄에 해당하는 값을 출력합니다.

---

## 입력 예제

```
2 5 4 1
```

---

## 출력 예제

```
116
```

---

## 예제 설명

2시 5분부터 116분이 흘러야 4시 1분이 됩니다.
