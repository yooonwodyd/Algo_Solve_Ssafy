# 알파벳 하나

- **정답률**: 100%  
- **총 제출 수**: 6회  

---

## 🧩 문제 설명

코디는 **N × N 격자**에 적힌 알파벳을 **하나의 알파벳으로 통일**하는 게임을 하고 있습니다.

초기 격자판의 각 칸에는 **대문자 알파벳 1개(A~Z)** 가 적혀 있습니다.  
게임에서는 **특정 칸의 알파벳을 다른 알파벳으로 바꾸는 연산**을 수행할 수 있으며,  
이 연산을 통해 격자 내 모든 칸의 알파벳을 동일하게 만들어 **하나의 알파벳만 남게** 하려고 합니다.

목표는 다음과 같습니다.

1. 가능한 **최소 연산 수행 횟수**를 구한다.
2. 그때 최종적으로 남게 되는 알파벳을 구한다.
   - 최종 알파벳 후보가 여러 개면 **사전순으로 가장 앞서는 알파벳**을 선택한다.

---

## 📥 입력

- 첫 번째 줄에 정수 **N**이 주어집니다.
- 두 번째 줄부터 **N개의 줄**에 걸쳐, 각 줄마다 길이 N의 **대문자 알파벳 문자열**이 공백 없이 주어집니다.  
  - (i행 j열)의 문자는 격자의 (i행 j열)에 적힌 알파벳을 의미합니다.

---

## 📤 출력

- 첫 번째 줄에 **최소 연산 수행 횟수**와 **최종적으로 남게 된 알파벳**을 공백으로 구분하여 출력합니다.

---

## ⚙️ 제한 조건

- \(3 \le N \le 100\)
- 각 칸에는 대문자 알파벳 `A` ~ `Z`가 주어집니다.
