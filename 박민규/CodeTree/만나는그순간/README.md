# 만나는 그 순간 (Easy)

- **XP**: 30XP  
- **평균 소요 시간**: 46분  
- **정답률**: 44%  
- **총 제출 수**: 4,656회  

---

## 🧩 문제 설명

A와 B는 **같은 시작점**에서 출발합니다.  
두 사람은 **1초에 1m**씩 이동하며, 각각 주어진 명령에 따라 **일정 시간 동안** 왼쪽 또는 오른쪽으로 이동합니다.

A와 B가 **움직이기 시작한 이후** 처음으로 다시 **같은 위치에서 만나게 되는 시간(초)** 이 몇 초 뒤인지 구하세요.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M**이 주어집니다.
2. 다음 **N줄**에는 A의 이동 정보 `(d, t)`가 공백으로 주어집니다.
3. 다음 **M줄**에는 B의 이동 정보 `(d, t)`가 공백으로 주어집니다.

각 `(d, t)`의 의미는 다음과 같습니다.

- `d`는 `L` 또는 `R`
  - `L`: 왼쪽으로 이동
  - `R`: 오른쪽으로 이동
- `t`초 동안 해당 방향으로 이동 (1초에 1m)

추가로 다음이 보장됩니다.

- A가 이동한 총 시간과 B가 이동한 총 시간은 **항상 동일**합니다.
- 처음 A와 B는 **서로 다른 방향으로만** 움직이기 시작한다고 가정해도 좋습니다.

---

## 📤 출력

- A와 B가 **처음으로 만나게 되는 시간(몇 초 뒤인지)** 을 출력합니다.
- 만약 두 사람이 **이동을 모두 마칠 때까지** 만나지 못하면 **-1**을 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N, M \le 1000\)
- \(1 \le t \le 1000\)
- \(d \in \{L, R\}\)

---

## 입력 예제 1

```
3 4
R 9
L 3
R 5
L 2
R 2
L 1
R 12
```

## 출력 예제 1
```
12
```
