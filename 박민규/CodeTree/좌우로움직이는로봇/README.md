# 좌우로 움직이는 로봇

- **XP**: 40XP  
- **평균 소요 시간**: 83분  
- **정답률**: 44%  
- **총 제출 수**: 3,341회  

---

## 🧩 문제 설명

1차원 직선 위에서 로봇 A와 B가 **1초에 1칸씩** 좌/우로만 이동합니다.

- 로봇 A의 이동 횟수: **N**
- 로봇 B의 이동 횟수: **M**

각 이동은 `(t, d)` 형태로 주어지며,
- `t`초 동안
- 방향 `d`로 이동합니다. (`d`는 `L` 또는 `R`)

두 로봇은 **처음에 같은 위치**에서 시작하지만,  
이 시작 상태에서의 “같은 위치”는 **횟수에 포함하지 않습니다**.

또한, 한 로봇이 모든 이동을 끝내면 그 이후에는 **그 위치에 계속 머무릅니다.**  
다른 로봇이 계속 움직이는 동안 두 로봇이 다시 같은 위치가 될 수도 있습니다.

단, **두 로봇이 모두 이동을 끝낸 이후의 시간**은 고려하지 않습니다.

---

## ✅ 세려는 경우(카운트 조건)

“바로 직전에는 서로 다른 위치였다가, 그 다음(다음 초)에는 같은 위치가 되는 경우”가  
총 몇 번 발생하는지를 구합니다.

즉, 어떤 초 `s`에 대해

- `posA(s-1) != posB(s-1)`
- `posA(s) == posB(s)`

이면 1번 카운트합니다.

---

## 📥 입력

1. 첫 번째 줄에 로봇 A와 B가 움직인 횟수 **N, M**이 공백으로 주어집니다.
2. 다음 **N줄**에 걸쳐 로봇 A의 이동 정보 `(t, d)`가 주어집니다.
3. 다음 **M줄**에 걸쳐 로봇 B의 이동 정보 `(t, d)`가 주어집니다.

---

## 📤 출력

- 조건을 만족하는 경우의 수를 첫 번째 줄에 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N, M \le 50{,}000\)
- 로봇 A와 B가 움직인 **총 이동 거리의 합은 1,000,000을 넘지 않음**을 가정해도 좋습니다.

---

```

```
