# Date to Date

2011년 m1월 d1일로부터  
2011년 m2월 d2일까지는 총 며칠이 있는지를 계산하는 프로그램을 작성해보세요.

2011년은 윤년이 아닌 해이기 때문에 2월은 28일까지 있습니다.  
단, 날짜 계산 시 **시작일을 포함하여** 세도록 합니다.

---

## 입력

첫 번째 줄에 m1, d1, m2, d2가 공백을 사이에 두고 주어집니다.

---

## 제한 조건

- 1 ≤ m1, m2 ≤ 12
- 1 ≤ d1, d2 ≤ 31
- 모순되는 경우는 없다고 가정해도 좋습니다.

---

## 출력

첫 번째 줄에 해당하는 값을 출력합니다.

---

## 입력 예제

```
11 5 12 21
```

---

## 출력 예제

```
47
```

---

## 예제 설명

11월 5일부터 11월 30일까지는 26일,  
12월 1일부터 12월 21일까지는 21일이 있기 때문에  
총 47일이 있습니다.
