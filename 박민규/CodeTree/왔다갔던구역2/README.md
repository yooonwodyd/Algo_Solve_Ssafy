# 왔다 갔다 구역 2 (Medium)

- **난이도**: Medium  
- **정답률**: 40%  
- **평균 소요 시간**: 65분  
- **총 제출 수**: 5,852회  

---

## 🧩 문제 설명

위치 0에서 시작하여 **N번의 명령**에 걸쳐 이동한 뒤,  
**2번 이상 지나간 영역의 크기**를 출력하는 프로그램을 작성하세요.

각 명령은 `"x L"` 또는 `"x R"` 형태로 주어집니다.

- `"x L"`: 왼쪽으로 x만큼 이동  
- `"x R"`: 오른쪽으로 x만큼 이동  

---

## 📥 입력

1. 첫 번째 줄에 정수 **N**이 주어집니다.  
2. 두 번째 줄부터 **N개의 줄**에 걸쳐 명령이 주어집니다.  
   - 각 줄은 `"x L"` 또는 `"x R"` 형태입니다.

---

## 📤 출력

명령을 주어진 순서대로 수행했을 때,  
**2번 이상 지나간 영역의 크기**를 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 100\)  
- \(1 \le x \le 10\)

---

## ✅ 입력 예제 1
```
6
2 R
6 L
1 R
8 L
1 R
2 R
```

---

## ✅ 출력 예제 1
```
6
```
