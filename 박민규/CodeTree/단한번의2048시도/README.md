# 단 한 번의 2048 시도

- **XP**: 90XP
- **평균 소요 시간**: 180분
- **정답률**: 55%
- **총 제출 수**: 1,487회

---

## 🧩 문제 설명

2048 게임은 **4 × 4 격자**에서 진행됩니다.  
상/하/좌/우 중 한 방향(`L`, `R`, `U`, `D`)을 선택하면, 격자의 모든 숫자가 그 방향으로 **한 번** 밀립니다.

이때 규칙은 다음과 같습니다.

### 1) 밀리기(중력 작용)

모든 숫자는 선택한 방향으로 가능한 한 끝까지 이동합니다. (빈 칸은 0)

### 2) 합쳐지기(병합)

이동 중 **같은 숫자**가 만나면 두 숫자가 하나로 합쳐집니다.  
합쳐진 숫자는 두 수의 합이 됩니다. (예: 2 + 2 = 4)

### 3) 연쇄 병합 금지

**한 번의 이동에서 이미 합쳐진 숫자는 다시 합쳐지지 않습니다.**  
즉, 한 번의 중력 작용 동안 어떤 타일은 **최대 1번만 병합**에 참여할 수 있습니다.

### 4) 3개 이상 연속일 때 병합 순서

같은 숫자가 3개 이상 연속으로 놓여 있고 해당 방향으로 밀리는 경우,
**벽(또는 바닥)에 더 가까운 숫자부터 2개씩만** 합쳐집니다.  
따라서 3개/4개가 한꺼번에 하나로 합쳐지는 일은 없고, 가까운 쌍부터 순서대로 병합됩니다.

---

## 📥 입력

- 1~4번째 줄: 각 행의 숫자 4개가 공백으로 주어집니다.
  - 숫자가 적혀있지 않은 칸은 `0`으로 주어집니다.
- 5번째 줄: 이동 방향을 나타내는 문자 `dir`가 주어집니다.
  - `dir ∈ {L, R, U, D}`
  - 각각 왼쪽, 오른쪽, 위, 아래 방향을 의미합니다.

---

## 📤 출력

- `dir` 방향으로 **한 번 이동한 이후의 결과 격자**를 4줄에 걸쳐 출력합니다.
- 각 줄에는 숫자 4개를 공백으로 출력합니다.
- 숫자가 없는 칸은 `0`을 출력합니다.

---

## ⚙️ 제한 조건

- 격자에 주어지는 수: \(0 \le \text{주어진 수} \le 2048\)
- (문제 설명상) 숫자는 0 또는 2 이상의 2의 거듭제곱 형태로 주어질 수 있습니다.

---

## ✅ 입력 예제 1

```
4 2 0 8
4 2 2 4
0 8 8 2
4 2 2 2
R
```

```
0 4 2 8
0 4 4 4
0 0 16 2
0 4 2 4
```
